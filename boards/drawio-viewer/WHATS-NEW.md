# What's New — DrawIO Viewer

One line per change, newest first. Keep it short. Record pending changes under a heading for
the **next version** you'll release (the version `board-manifest.json` will be bumped to) — so
the number is decided up front and nothing needs renaming at release time.

## 1.0.3

- Toolbar chrome now follows Persephone's own theme chrome color (Persephone 4.0.16+; unchanged look on older versions), with softer hover highlights.

## 1.0.2
- Added a Save menu — export the current page as SVG (vector) or PNG (2×) to a file.
- Added zooming and panning.
- Diagram canvas is now always white (renders diagrams faithfully, independent of app theme).
- Added "Open in Drawing Editor" — opens the diagram as an editable Excalidraw drawing.
